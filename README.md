# Smart Campus Management System

A full-stack Smart Campus Management System built using:

* Java Spring Boot
* React + Vite
* MySQL
* JWT Authentication
* Tailwind CSS

## Features

### Admin Module

* Manage students
* Manage attendance
* Manage assignments
* Manage timetable
* Manage announcements
* Review leave requests

### Student Module

* Student dashboard
* View attendance
* View timetable
* View assignments
* Apply leave requests
* Edit profile
* View announcements

## Tech Stack

### Backend

* Spring Boot
* Spring Security
* JWT
* Hibernate/JPA
* MySQL

### Frontend

* React
* Vite
* Tailwind CSS
* Axios

## Setup Instructions

### Backend

1. Configure MySQL database
2. Update `application.properties`
3. Run:

```bash
mvn spring-boot:run
```

Backend runs on:

```text
http://localhost:8080
```

Swagger:

```text
http://localhost:8080/swagger-ui/index.html
```

---

### Frontend

Install dependencies:

```bash
npm install
```

Run frontend:

```bash
npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```
